# Smart Waste Management System

Smart Waste Management System is a digital platform that allows citizens to submit garbage pickup requests, track request status using a unique tracking ID, and raise complaints for unresolved issues.  
The system improves transparency, communication, and efficiency in urban waste management while helping maintain clean and hygienic surroundings.
